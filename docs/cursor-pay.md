## Cursor Team Plan 计费机制

### 1. 基础订阅费用：$40/月/人（固定费用）

- 无论使用多少，每月都会收取 $40
- 这笔费用包含 $20 的使用额度
- 即使不使用，也需要支付 $40

### 2. 包含额度：$20/月（已包含在 $40 中）

- 这 $20 已包含在 $40 订阅费中
- 不是额外费用，是订阅费的一部分
- 如果使用不超过 $20，总费用就是 $40

### 3. 按需计费：超出 $20 的部分（额外费用）

- 只有超出 $20 的部分才会额外计费
- 按实际使用量计费
- 上限：每人每月最多额外 $250

### 4. 费用计算示例

**场景 1：使用量 ≤ $20**
- 订阅费：$40
- 使用费用：$0（已包含在订阅费中）
- 总费用：$40

**场景 2：使用量 = $25**
- 订阅费：$40
- 包含额度：$20（已包含）
- 超出部分：$5（额外计费）
- 总费用：$40 + $5 = $45

**场景 3：使用量 = $300**
- 订阅费：$40
- 包含额度：$20（已包含）
- 超出部分：$250（达到上限）
- 总费用：$40 + $250 = $290

### 5. 总结

| 费用类型 | 金额 | 说明 |
|---------|------|------|
| 基础订阅费 | $40/月 | 固定费用，无论使用多少都会收取 |
| 包含使用额度 | $20/月 | 已包含在 $40 中，不是额外费用 |
| 按需计费 | $0 - $250/月 | 只有超出 $20 后才计费，按实际使用量 |

**简单来说：**
- $40/月是固定费用，必须支付
- $20 已包含在 $40 中
- 超出 $20 的部分会额外计费，最多 $250
